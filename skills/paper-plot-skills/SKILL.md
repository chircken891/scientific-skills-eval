# Paper Plot Skills - Figure Generation

## Overview
Paper Plot Skills provides 9 types of real paper figures with plot-from-image capability for generating publication-quality scientific visualizations.

## When to Use
- Creating figures for research papers
- Generating charts from experimental data
- Making visualizations for presentations
- Preparing publication-ready plots

## Process
1. Define visualization requirements and data source
2. Select appropriate plot type
3. Generate figure with publication-quality styling
4. Export in journal-compatible formats
