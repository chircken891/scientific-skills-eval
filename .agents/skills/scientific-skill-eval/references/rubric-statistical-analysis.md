# Rubric: Statistical Data Analysis

For skills that perform hypothesis testing, regression analysis, statistical modeling, and integrate with R/Python for computation.

## Score Criteria

| Score | Criteria |
|-------|----------|
| 5/5 | Comprehensive statistical analysis: hypothesis testing (t-test, chi-square, ANOVA, Mann-Whitney), regression (linear, logistic, Cox, mixed-effects), survival analysis (Kaplan-Meier, Cox PH), meta-analysis (fixed/random effects, meta-regression, publication bias assessment). R/Python integration with key analysis packages (metafor, lme4, survival, statsmodels, scipy). Automated assumption checking (normality, homoscedasticity, multicollinearity). Results reported with effect sizes, confidence intervals, and interpretation. |
| 4/5 | Core statistical tests with regression capabilities. R/Python integration for common analyses. Basic assumption checking. Results with point estimates and CIs. |
| 3/5 | Basic statistical tests available (t-test, chi-square, linear regression). Limited R/Python integration. Manual assumption verification. Results reported but may need statistical review. |
| 2/5 | Can perform only basic descriptive statistics (mean, median, SD). No inferential statistics. No integration with statistical computing environments. |
| 1/5 | Cannot perform statistical analysis beyond simple counting/listing. No statistical methodology awareness. |

## Score Translation

- **5** → 完整统计工具箱+生存分析+Meta分析+R+Python集成+假设检验+效应量+CI
- **4** → 核心检验+回归+R/Python集成，但假设检验部分不够全
- **3** → t检验/卡方/线性回归可用，但集成度和自动检验不足
- **2** → 仅描述性统计
- **1** → 无统计能力
