# Rubric: Literature Review / Systematic Review

For skills that perform structured review workflows, synthesize findings across sources, and produce formatted reviews.

## Score Criteria

| Score | Criteria |
|-------|----------|
| 5/5 | Full systematic review workflow: PRISMA-compliant screening, abstract/full-text filtering, risk of bias assessment, evidence table generation. Database integration with PubMed Central, Cochrane, etc. Thematic synthesis with citation network analysis. Auto-generates PRISMA flow diagram. Exports to structured review formats. |
| 4/5 | Structured review with title/abstract screening and data extraction. Supports inclusion/exclusion criteria. Basic quality assessment. Thematic grouping of findings. Exports summary tables. |
| 3/5 | Literature summary with key finding extraction. Basic categorization by theme. Can answer specific review questions. No systematic methodology adherence. |
| 2/5 | Simple literature list with one-sentence summaries per paper. No extraction, no quality assessment, no synthesis. |
| 1/5 | Cannot perform structured literature review. May list papers without meaningful synthesis or analysis. |

## Score Translation

- **5** → PRISMA全流程合规+数据库集成+证据表+PRISMA流程图自动生成
- **4** → 有结构化筛查和数据提取，但缺自动PRISMA图或方法学不够系统
- **3** → 有人工质量评估但无PRISMA合规要求
- **2** → 论文列表+摘要，无质量评估和综合
- **1** → 无法执行结构化综述

## Function Type ID

`literature-review`
