# Rubric: Research Tool / Framework

For skills that function as modular research libraries, multi-agent orchestrators, or reusable pipeline frameworks with broad domain coverage.

## Score Criteria

| Score | Criteria |
|-------|----------|
| 5/5 | Modular skill/tool library with 50+ focused capabilities across 8+ research domains. Multi-agent orchestration with role-based delegation. Reusable pipeline components with documented interfaces. Extensive domain coverage (literature, analysis, writing, submission, peer review). Plugin architecture allowing external tool integration. Versioned components with upgrade path. |
| 4/5 | Skill library with 20-50 capabilities across 5-7 domains. Basic agent orchestration. Reusable components documented for most modules. Good domain coverage with some gaps. |
| 3/5 | 10-20 capabilities across 3-4 domains. Monolithic rather than modular. Some reusable patterns but not systematically documented. Limited orchestration support. |
| 2/5 | 5-10 capabilities across 1-2 domains. Tightly coupled code with no modularity. No orchestration. Documentation minimal. |
| 1/5 | Fewer than 5 capabilities or single-use scripts. No modularity. No documentation. Framework claims not substantiated by structure. |

## Score Translation

- **5** → 50+能力+8+领域+多agent编排+插件架构+版本管理
- **4** → 20-50能力但编排/模块化某处不足
- **3** → 10-20能力，单体架构
- **2** → 5-10能力，紧耦合
- **1** → <5能力或框架名不副实

## Function Type ID

`research-tool`
