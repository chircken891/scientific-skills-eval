# Rubric: Journal-Specific Skills

For skills that handle formatting and compliance for specific journals (Nature, Science, Cell, etc.), including reviewer response and publication workflow.

## Score Criteria

| Score | Criteria |
|-------|----------|
| 5/5 | Full journal-specific compliance for Nature/Science/Cell families. Manuscript formatting per journal guidelines (structure, length, reference style, figure limits). Reviewer response template generation (point-by-point, with change tracking). Submission-ready package preparation (cover letter, highlights, graphical abstract). Journal-specific writing style adaptation (tone, terminology, structure per journal culture). Format validation against latest author guidelines. |
| 4/5 | Multi-journal support (3+ journals) with formatting templates. Reviewer response guidance. Cover letter generation. Style adaptation for target journal. |
| 3/5 | Single journal support (e.g., Nature only) with basic formatting. General reviewer response advice. Basic cover letter template. |
| 2/5 | Generic journal formatting guide (no specific journal adaptation). General publication advice. Cannot validate against journal guidelines. |
| 1/5 | No journal-specific capability. Cannot distinguish between journal requirements or produce formatted output. |

## Score Translation

- **5** → Nature/Science/Cell全合规+审稿回复模板+投稿包+格式自动验证
- **4** → 3+期刊但投稿包或验证不全
- **3** → 单期刊+基础格式+通用审稿建议
- **2** → 通用格式建议，无期刊特定适配
- **1** → 无期刊专长

## Function Type ID

`journal-specific`
