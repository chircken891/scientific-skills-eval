# Rubric: Academic Writing / Paper Writing

For skills that assist with academic manuscript preparation, including LaTeX/Typst support, journal-adaptive style, multi-round revision, and review simulation.

## Score Criteria

| Score | Criteria |
|-------|----------|
| 5/5 | Full manuscript generation with LaTeX/Typst support. Journal-adaptive writing style (templates for Nature, Science, PLOS, BMJ, etc.). Multi-round revision with change tracking. Automated review simulation (peer review, statistical review). Section-specific optimization (abstract, introduction, methods, results, discussion, conclusion). Reference formatting per journal style. |
| 4/5 | Manuscript generation with LaTeX support. Supports 3+ journal templates. Multi-round revision support. Basic review checklist. Good section-level writing. |
| 3/5 | Manuscript drafting with basic structure. Single journal template. Limited revision support. Generic academic tone without journal adaptation. |
| 2/5 | Basic academic paragraph generation. No LaTeX/journal awareness. No revision workflow. Produces passable but generic academic text. |
| 1/5 | Cannot produce structured academic writing. Output reads as casual/general-purpose text. No awareness of academic conventions. |

## Score Translation

- **5** → 全流程LaTeX+多期刊模板+多轮修订+审稿模拟+格式自动合规
- **4** → 3+期刊模板但审稿模拟不全或某环节较弱
- **3** → 单期刊+基础修订支持，学术语气通用不精准
- **2** → 无LaTeX/期刊感知，仅通用学术段落
- **1** → 非学术级内容
