## New Ranking System (Not Weighted Sum)

**Sort by:**
1. **Security verdict** — VETOED repos excluded from recommendations
2. **Threshold category** — AUTO-RECOMMEND (>4.0) > CANDIDATE (3.0-4.0) > EXCLUDE (<3.0)
3. **DepthScore descending** — within same threshold category
4. **Integration_小计 descending** — within same DepthScore
5. **Coverage stages** — more stages = better (final tiebreaker)

## Recommendation Rules
- Only security APPROVED repos enter recommendations
- Prioritize AUTO-RECOMMEND over CANDIDATE
- Cover diverse research stages (literature/analysis/writing/submission)
- Medical research fit (epidemiology/biostatistics/oncology) prioritized
- Higher Integration_小计 within same threshold = better combinability
