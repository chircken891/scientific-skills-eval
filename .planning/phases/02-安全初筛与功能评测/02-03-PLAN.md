## New Ranking System (Not Weighted Sum)

**The old system ranked by weighted total (100pt). The new system ranks by:**

1. **Security verdict first** — VETOED repos are excluded from recommendations
2. **Threshold category second** — AUTO-RECOMMEND (>4.0) > CANDIDATE (3.0-4.0) > EXCLUDE (<3.0)
3. **DepthScore third** — Within same threshold category, sort by DepthScore descending
4. **Integration score fourth** — Within same DepthScore, use Integration_小计 as tiebreaker (higher = better combinability)
5. **Coverage descriptions fifth** — For final tiebreaker, use coverage stage coverage (more stages = better)

**Do NOT use alphabetical ordering as a tiebreaker.**
