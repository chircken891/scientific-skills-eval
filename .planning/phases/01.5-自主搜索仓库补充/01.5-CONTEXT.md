<decisions>
## Implementation Decisions

### 搜索范围与平台
- **D-01:** 跨平台全局搜索 — 不限于 GitHub，覆盖所有平台：GitHub、npm registry、VS Code marketplace、Cline/MCP registry 等所有能想到的平台

### 搜索深度
- **D-02:** 全面彻底 — 每个平台细挖，关联仓库、评分、维护状态

### 收录标准
- **D-03:** 广义科研工具 — 包括 AI4Science、数据科学、计算社会科学等，跨学科复合型工具也纳入范围

### 去重策略
- **D-04:** 去重 — 同一作者同名仓库只保留一个最新版本
- **D-05:** 质量优先 — 去重时按 star 数量、活跃度、维护状态选择

### 更新机制
- **D-06:** 一次性补充 — Phase 1.5 本次一次性搜索
- **D-07:** 保留更新窗口 — 留一个机制接口，后续如有新来源可继续补充更新 SKILLS-INVENTORY.md，不重新走完整讨论流程

### Claude's Discretion
无 — 所有决策均已由用户明确锁定
